# TodoList

A simple and interactive Todo List web application built with JavaScript. This project helps users organize and keep track of their daily tasks efficiently.

## Features

- Add new tasks to the list
- Check off completed tasks with a single click
- Delete tasks individually
- Responsive, clean, and easy-to-use interface
- All design and logic generated dynamically with JavaScript
